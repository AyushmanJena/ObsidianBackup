Pointcut expressions
`@Before("execution(* com.luv2code.aopdemo.dao.*.*(..))")`
// any return type, any class, any method in the given package




In ordering aspects
note : after placing all the aspects in different classes you need to use full package names to access other pointcut expressions from other classes
: 
```java
@Aspect  
@Component  
public class MyCloudLogAsyncAspect {  
    @Before("com.luv2code.aopdemo.aspect.LuvAopExpressions.forDaoPackageNoGetterSetter()")  
    public void logToCloudAsync(){  
        System.out.println("\n>>> Logging to Cloud in async fashion");  
    }  
}
```
