What is Ionic ?
Ionic is a set of tools that use html, css and js
for creating gui 
made for frontend
make applications that can run on web, android and ios

So we can have same source code for multiple platforms, this is called hybrid development.

You can build ionic apps with Angular, React or Vue

### Installation

npm install -g @ionic/cli

Create ionic project : 
`ionic start project-name`

Ionic gives you 3 templates to choose from : 
blank
tabs
side menu

To run project : `ionic serve`



### Project Structure : 
MOST FILES ARE SIMILAR TO ANGULAR
ionic.config.json : project details
karma.conf.js : to run tests config file
